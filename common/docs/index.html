@hpke/common

A TypeScript Hybrid Public Key Encryption (HPKE) internal-use common module for @hpke family modules.

Installation

Do not install this package directly, it is an internal package used by @hpke/core, @hpke/chacha20poly1305 and so on. Use either of them instead.

Contributing

We welcome all kind of contributions, filing issues, suggesting new features or sending PRs.
